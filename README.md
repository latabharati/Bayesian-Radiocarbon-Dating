# Bayesian-Radiocarbon-Dating
Implemented a Metropolis-within-Gibbs sampler to estimate calendar ages from radiocarbon measurements using the IntCal20 calibration curve.
Applied Bayesian inference with ordering constraints to model five archaeological samples and generate posterior age estimates with 95% credible intervals.
Assessed convergence using trace plots, running means, autocorrelation plots, Effective Sample Size and Monte Carlo error.
Demonstrated strong understanding of simulation methods, uncertainty quantification and statistical modelling in R.
